# Frontend Project 12

Чат-приложение на React с серверной частью.

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Сборка фронтенда
make build

# Запуск сервера
make start
```

## Технологии

- React + Vite
- @hexlet/chat-server
- Проксирование API запросов
